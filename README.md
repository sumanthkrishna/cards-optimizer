# Cards Optimizer

React Native Expo prototype for a stacked credit-card wallet UI.

## Features

- Stacked card layout
- Tap a card to reveal a full-size view
- Generic placeholder card art for Chase, Discover, Citi, Wells Fargo, and American Express-style samples
- Balance and credit utilization panel

## Run locally

```bash
npm install
npm start
```

Then open with Expo Go or an emulator.

## Note

This project uses generic placeholder card images. Replace them only with artwork you have rights to use.
